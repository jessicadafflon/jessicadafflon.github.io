Implementation of Large-scale Brain Simulations on NeuroML

Jan 1, 0001
PDF WilsonandCowan delMolino MejiasEtAl2016

This project aimed to implement Large-scale Brain Simulations, which are models that simulate the interaction between different brain areas, in NeuroML. NeuroML is a tool that standardises commonly used parameters to describe neuroscience models. This standardisation is essential for making computational models used in neuroscience easily shareable and reproducible. I have implemented and open-sourced three different models: 1) Wilson and Cowan, 2) delMolino, and 2) MejiasEtAl2016. A detailed description of the analysis can be found on the pdf linked to this page, which was presented at the CNS 2019 in Barcelona. With this project, I learned how to use numba, intensively used git and learned how to collaborate on an open science project.
